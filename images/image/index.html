Steps in Creating a Mappable Image

  1. Pick an Image
  2. Check its folder properties
  3. Hard code Dimensions
  4. Record URL and Dimensions in blockquote, cite, and comments for date accessed and origonal dimensions
  5. Add usemap attribute, map tag linked to the usemap class
  6. Add area tags with shape, coords, alt, href , and target attributes

Other shapes in a mapped image are possible too ...

  • Polygon might be the most useful here since rectangles and circles do not cover the frames well.
  • Change the image to include plygons that are more defined of the shape
  • Hint: use a paper map first and record coordiantes with all the clues of what your browser is showing
